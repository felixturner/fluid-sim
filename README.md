# fluid-sim

2D Fluid Simulation using Three.js. 

Forked from https://github.com/amsXYZ/three-fluid-sim

Updates:

- Migrated to Vite
- Updated to THREE.js r180
- Removed TypeScript
- Optimized

## Commands

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Deployment Build

```bash
npm run build
```
